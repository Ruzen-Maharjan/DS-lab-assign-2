# DS-lab-assign-2
Statistics and Trends

#Overview
This report analyzes the relationship between C02 emissions, Population and Agricultural Land using World Bank Data.
The main goal of this assignment is to understand trends overtime and examine hoe different factors and indicators influence C02 emissions in the selected countries
# Data source 
The data used in this project is taken from World Bank Open Data:

- CO₂ Emissions: https://data.worldbank.org/indicator/EN.GHG.CO2.MT.CE.AR5  
- Population: https://data.worldbank.org/indicator/SP.POP.TOTL  
- Agricultural Land: https://data.worldbank.org/indicator/AG.LND.AGRI.K2  

# Python methods used for this report
- Data cleaning and preprocessing
- Correlation analysis
- Rate of change calculation
- Bootstrapping
- Data visualization using multiple graphs like line graphs, scatter plot, bar chart.
